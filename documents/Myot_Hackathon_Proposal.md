# Myot — Make Your Own TV

> **"TV를 보는 시대에서, TV를 만드는 시대로"**

---

## 1. Problem Statement

### 현재 스마트 TV의 한계

스마트폰과 스마트워치는 이미 사용자가 홈 화면을 자유롭게 구성한다. 위젯을 배치하고, 테마를 바꾸고, 워치페이스를 마켓에서 구매한다. 그런데 **스마트 TV는 여전히 제조사가 정해준 화면을 그대로 사용**하고 있다.

| | 스마트폰 | 스마트워치 | 스마트 TV |
|---|---|---|---|
| 홈 화면 커스텀 | ✅ 완전 자유 | ✅ 워치페이스 | ❌ 고정 UI |
| 위젯 배치 | ✅ 드래그 앤 드롭 | ✅ 컴플리케이션 | ❌ 없음 |
| 테마 마켓 | ✅ 활발 | ✅ 활발 | ❌ 없음 |
| 자연어 커스터마이징 | ❌ 없음 | ❌ 없음 | ❌ 없음 |

**핵심 문제:** TV 앞의 사용자는 리모컨 하나로 모든 것을 해결해야 하지만, 기존 UI 커스텀 방식(드래그 앤 드롭)은 리모컨에 적합하지 않다.

---

## 2. Solution: Myot (마이엇)

### 핵심 컨셉 — "Vibe Decorating"

바이브 코딩이 자연어로 코드를 만들듯, **Myot은 자연어로 TV 홈 화면을 만든다.**

기존 스마트 TV의 위젯은 제조사가 미리 만들어 제공하는 **"고정된 틀"**이다. Myot은 다르다. 사용자가 말한 요구사항을 AI가 해석해서 **위젯 자체를 실시간으로 생성**한다.

```
👤 "왼쪽에 아내 사진 크게 넣고, 오른쪽에 오늘 날씨랑 일정 보여줘"
🤖 → Google Photos에서 최근 아내 사진 선택
   → 왼쪽 60% 영역에 사진 위젯 생성
   → 오른쪽 상단에 날씨 위젯, 하단에 캘린더 위젯 배치

👤 "블랙 테마로 바꾸고 투명도 50%로 해줘"
🤖 → 전체 배경 #1A1A1A 적용, 위젯 opacity 50% 조정

👤 "추천 레이아웃 3개 보여줘"
🤖 → 사용자 데이터 분석 → 3가지 레이아웃 순차 프리뷰
```

## 3. 차별화 포인트

### 3.1 AI-Generated Widgets (핵심 차별화)

| 기존 스마트 TV | Myot |
|---|---|
| 미리 만들어진 위젯 선택 | 자연어로 위젯 자체를 생성 |
| 제조사 제공 레이아웃 | AI가 맥락에 맞는 레이아웃 생성 |
| 리모컨으로 수동 배치 | 음성으로 즉시 반영 |

**예시:** "주식 위젯 넣어줘"가 아니라 **"내가 가진 삼성전자, 애플 주식만 보여주고, 5% 이상 변동 시 빨간색으로 알려줘"** — 이 요구사항에 맞는 커스텀 위젯이 즉석에서 만들어진다.

### 3.2 Context-Aware Ambient Screen

TV는 항상 켜져 있는 디스플레이다. Myot은 이를 활용한다:

- **시간 기반:** 아침엔 뉴스+날씨+일정 → 저녁엔 OTT 추천+가족사진
- **인물 인식:** 카메라(옵션)로 시청자 감지 → 아빠가 앉으면 주식+스포츠, 아이가 앉으면 학습콘텐츠+친구통화
- **외부 연동:** 비 오는 날 → 자동으로 실내 운동 추천 위젯 + 따뜻한 톤 배경

### 3.3 Living Widget — 운동 연동 시나리오

스마트워치/Google Location History와 연동하여 TV를 개인 코치로 활용:

```
📺 [홈 화면에 오늘의 러닝 경로가 Google Map으로 표시됨]

👤 "이 정도면 가뿐한데? 더 늘려도 되겠어"

🤖 "오, 대단하시네요! 그럼 1km만 늘려 볼까요?
    
    🥇 첫 번째 추천: 한강뷰 코스
       → 벚꽃 시즌 최적 루트, +120 kcal
    
    🥈 두 번째 추천: 카페 투어 코스  
       → 유명 커피숍 3곳 경유, +95 kcal
    
    🥉 세 번째 추천: 피톤치드 코스
       → 숲길이지만 오르막 있음, +180 kcal
       ⚠️ 심박수 평균 155bpm 이상이면 페이스 조절 추천

    어떤 코스로 내일 달려볼까요?"
```

### 3.4 Generational Bridge — 세대 연결 위젯

노년층을 위한 특화 기능:

- **패밀리 포토 월:** 가족 공유 앨범 자동 슬라이드쇼
- **원버튼 영상통화:** 손주 얼굴 사진 터치 → 즉시 영상통화
- **건강 대시보드:** 혈압, 복약 시간, 오늘 걸음수를 큰 글씨로
- **AI 말벗:** "오늘 뭐 했어?" → TV가 대화 상대가 되어주는 동반자 모드
- **자연어 음성 설정:** "글씨 좀 더 크게 해줘", "이 사진 계속 보여줘" — 터치 없이 음성만으로

## 4. 기능 상세

### 4.1 Natural Language UI Engine

| 자연어 명령 | 실행 결과 |
|---|---|
| "유튜브 아이콘을 오른쪽 위로 옮겨줘" | 앱 아이콘 위치 변경 |
| "배경을 우리 가족사진으로 바꿔줘" | Google Photos 연동, 배경 설정 |
| "캘린더 위젯 크기를 두 배로 키워줘" | 위젯 리사이즈 |
| "지금 화면을 저장해줘" | 레이아웃 템플릿 저장 |
| "주말용 레이아웃 따로 만들어줘" | 시간 기반 레이아웃 자동 전환 |

### 4.2 위젯 카테고리

- **앱 런처:** YouTube, Netflix, TV+, Disney+, 사용자 앱
- **정보 위젯:** 날씨, 시계, 뉴스, 주식, 환율, 미세먼지
- **Google 연동:** Calendar, Photos, Maps, Keep, Gmail 알림
- **건강/운동:** 오늘 활동량, 러닝 경로, 수면 데이터, 심박수 추이
- **스마트홈:** 조명 제어, 온도 조절, CCTV 미리보기, 로봇청소기 상태
- **소셜:** 가족 공유 사진, 메시지 알림, 영상통화 바로가기
- **AI 생성:** 사용자 요청에 따라 즉석에서 만들어지는 맞춤형 위젯

### 4.3 Template Marketplace

스마트폰 테마 스토어, 워치페이스 마켓과 동일한 생태계:

- **디자이너가 만든 프리미엄 레이아웃** 판매
- **사용자 생성 레이아웃** 공유/판매 (UGC)
- **상황별 큐레이션:** 게이머용, 재택근무용, 키즈용, 시니어용
- **시즌 이벤트:** 크리스마스 테마, 월드컵 특별 레이아웃
- **AI 추천:** "당신의 사용 패턴에 맞는 레이아웃을 추천합니다"

## 5. 추가 차별화 아이디어

### 5.1 Multi-Screen Orchestration

TV 하나가 아닌, 집 안 모든 화면의 통합 관리:

- 거실 TV + 주방 디스플레이 + 침실 TV를 하나의 캔버스로
- "주방에 레시피 띄우면서 거실엔 타이머 보여줘"
- 폰에서 레이아웃 미리보기 → TV에 원터치 적용

### 5.2 Mood Sync

음악/조명/TV 배경의 통합 분위기 제어:

- "영화 모드" → 조명 어둡게 + TV 배경 블랙 + 사운드바 서라운드
- "파티 모드" → 스포티파이 연동 + 네온 컬러 배경 + IoT 조명 싱크
- "집중 모드" → 알림 차단 + 타이머 위젯 + 최소 UI

### 5.3 Digital Legacy

가족 기록을 TV에 담아 세대를 잇는 기능:

- 가족 타임라인 위젯: 결혼, 출산, 졸업 등 가족사를 슬라이드로
- "오늘 같은 날" 위젯: 1년 전, 5년 전 오늘의 가족 사진/영상
- 할머니의 레시피 위젯: 음성으로 녹음한 요리법을 AI가 정리

### 5.4 Emergency Widget

긴급 상황 대응을 위한 스마트 위젯:

- 지진/태풍 경보 시 → 전체 화면 긴급 알림 + 대피 경로 지도
- 노인 독거 모드 → 일정 시간 TV 조작 없으면 가족에게 알림
- 영유아 모드 → 시청 시간 자동 관리 + 휴식 알림

## 6. 기술 아키텍처

```
┌─────────────────────────────────────────────────┐
│                  Myot Platform                   │
├─────────┬──────────┬──────────┬─────────────────┤
│  NLU    │  Widget  │  Layout  │  Template       │
│  Engine │  Factory │  Engine  │  Marketplace    │
├─────────┴──────────┴──────────┴─────────────────┤
│            Context Manager                       │
│  (Time / User / Device / Location / Health)      │
├─────────────────────────────────────────────────┤
│            Integration Layer                     │
│  Google APIs │ OTT APIs │ SmartHome │ Wearable  │
└─────────────────────────────────────────────────┘
```

- **NLU Engine:** 자연어 → 위젯 생성/배치 명령 변환 (LLM 기반)
- **Widget Factory:** 요구사항 기반 위젯 동적 생성 (Code Generation)
- **Layout Engine:** 충돌 방지, 반응형 배치, 다중 해상도 대응
- **Context Manager:** 사용자 맥락 수집 및 실시간 반영
- **Template Marketplace:** 레이아웃 거래 및 공유 플랫폼
